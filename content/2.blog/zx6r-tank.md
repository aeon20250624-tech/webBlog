# ガソリンタンク交換 ZX-6 ZX636

わりあい綺麗なガソリンタンク（ZX-6）がヤフオクにでていたので、落札した。  
&nbsp;  
早速、取りつけようとしたら、タンクキャップが合わない。  
過去の落札情報とか確認してみるとどうも年式違いだったようだ。  
商品説明にはZX636E型とあったのだが、ZX636G用だったようだ。  
&nbsp;  
落札したタンク  
![new Tank](/img/IMG_0769.jpg){.article-image}!  
&nbsp;  
交換前のタンク  
![new Tank](/img/IMG_0768.jpg){.article-image}!  
&nbsp;  
落札タンク側はネジ4本で固定するが  
交換前タンクの方は３本だ。  
給油口の大きさも違う。落札タンクの方が小さくなっている。  
これはコストダウンの一環だろうか？  
&nbsp;  
コックのデザインも交換前の方がレーシーな感じがする。  
&nbsp;  
落札したタンク  
![new Tank](/img/IMG_0773.jpg){.article-image}!  
&nbsp;  
交換前のタンク  
![new Tank](/img/IMG_0777.jpg){.article-image}!  
&nbsp;  
売主に連絡して返品返却を頼んで快諾してくれたのだが、  
商品自体が安かったしキャップだけ交換すればこのまま使えそうだったので  
値引き交渉して半額にしてもらった。  
&nbsp;  
キャップの方は鍵なしを1300円でゲットし、  
ちょっと分解して、カギ穴にタガネをブチこんで穴を拡げて  
メガネレンチが使えるマイナスドライバー突っ込んで力づくで回したところ、  
交換前タンクキャップの鍵でも回るようになった。  
たぶん、厚みが合えばどんな鍵でも回りそうだ。  
&nbsp;  
中免時代に初めて買ったZ400GPがアイスの棒でもキャップを開けられたのを思い出した。  
当時は、 **さすが男カワサキ** 、と仲間内で笑った記憶がある。  
&nbsp;  
タンクの車体への固定、ポンプのタンクへの固定は問題なかった。  
&nbsp;  
結局、これが  
![new Tank](/img/IMG_0778.jpg){.article-image}!  
&nbsp;  
こうなった  
![new Tank](/img/IMG_0772.jpg){.article-image}!  
&nbsp;  
キレイになったけど、以前のヘコミがなにか落ち葉が付いているようにも見えるといわれていたので  
なくなったらなくなったで、なにか寂しい気もする。  
（ゼッケン26がついているのでハッパ２６とか呼ばれていた）  
アト、ちっちゃくなった気もするが、これは気のせいだろう。
